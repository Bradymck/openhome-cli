# Volume Control

Control your OpenHome DevKit speaker volume with voice commands.

## Trigger Words

- "volume"
- "turn up"
- "turn down"
- "louder"
- "quieter"
- "mute"
- "unmute"
- "raise volume"
- "lower volume"
- "volume control"

## Commands

- **"raise" / "louder" / "turn up"** — increase volume by 10%
- **"lower" / "quieter" / "turn down"** — decrease volume by 10%
- **"mute" / "silent"** — set to 0
- **"max" / "full"** — set to 100
- **"set to 50"** — set to a specific percentage
